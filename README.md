# Municipal Waste Management Analytics (Power BI)

## Overview
This project analyzes municipal waste generation, recycling efficiency, and population-adjusted waste pressure across Indian cities (2019â€“2023).

The objective is to understand whether awareness campaigns translate into better waste outcomes and identify data-driven intervention priorities.

## Tools & Skills
- Power BI (DAX, Data Modeling)
- KPI Design & Segmentation
- Trend & Comparative Analysis
- Dashboard Storytelling

## Key Insights
- Awareness campaigns alone do not consistently reduce waste generation
- Tier-2 cities show higher density-adjusted waste pressure than some metros
- Operational execution plays a larger role than population density alone

## Business Impact
- Enables prioritization of high-risk cities
- Supports targeted waste management interventions
- Helps improve ROI on sustainability programs
